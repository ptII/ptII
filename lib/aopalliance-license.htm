License for aopalliance.jar

AOP Alliance is used by the HandSimDroid project to provide a Ptolemy II user interface on platforms like the Android. The Google Guice depends on this jar at runtime. The tests in $PTII/ptserver/test/junit use Guice.

The website for AOP Alliance is http://aopalliance.sourceforge.net/.

As of June, 2011, the license from http://aopalliance.sourceforge.net/.said:

The AOP Alliance project is a joint open-source project between several software engineering people who are interested in AOP and Java. LICENCE: all the source code provided by AOP Alliance is Public Domain.

Ptolemy II Design Document

The Ptolemy II Design Document has three parts:

  • A tutorial explaining how to build and use Ptolemy II models.
  • A detailed description of the core software architecture.
  • Descriptions of each of the domains.
The latest version of the Ptolemy II design doc can always be found at http://ptolemy.eecs.berkeley.edu/ptolemyII/designdoc.htm

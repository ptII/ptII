Below are the steps necessary to instal the Oracle Berkeley XML DB on Mac.
  1. Go to http://www.oracle.com/technology/software/products/berkeley-db/xml/index.html.
  2. Download the Berkeley DB XML 2.5.16.tar.gz (41M) from that page http://www.oracle.com/technology/software/products/berkeley-db/htdocs/popup/xml/2.5.16/xml-targz.html.
  3. Unzip the downloaded tar.gz file in your intended installing folder for the XML DB. $PTII/vendors/misc is a good location: 
    cd $PTII/vendors/misc
tar -zxf ~/Downloads/dbxml-2.5.16.tar.gz
  4. Go to the folder that was created and execute buildall.sh.
    Note that the build will take 40-50 minutes. 
    cd dbxml-2.5.16
sh buildall.sh >& build.out &
tail -f build.out
    Please just wait till the end to see the successfully installation message. It only takes such a long time once for the first time installation.
  5. After the build is done, go to the folder your_installation_path/dbxml-2.5.16/install/bin, and execute the command dbxmlyour_installation_path/dbxml-2.5.16/install/bin/dbxml.
  6. Now you should see the prompt of dbxml appears. You can test whether the XMLDB runs properly by testing with the guidance from your_installation_path/dbxml-2.5.16/dbxml/docs/intro_xml/running.html.

PNButterfly

This page contains an applet that should appear as a separate window.

Author: Man-Kit Leung The model traces an elaborate curve called the Butterfly curve, described by T. Fay (1989). The model is equivalent to $PTII/ptolemy/domains/sdf/demo/Butterfly/Butterfly.xml, except that the math Expression actor is converted to several atomic math actors and the SDFDirector is replaced by the Process Network (PN) Director. The generated code executes each actor in a separate thread and communicate with others through synchronized buffers.

ControlRobot

This page contains an applet that should appear as a separate window.

This model illustrates the concurrent control in manipulating the behavior of the iRobot. Through invoking the CodeGenerator, this model can be directly transformed into executable code that runs on the OpenRTOS+Luminary target, which in turn interface with the iRobot through a serial connection. Generating multitasking code can overlap computation with I/O time, which greatly benefits performance and doing more sophisticated control. Author: Man-Kit Leung

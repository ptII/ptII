SliderParameter

SliderParameter is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details. This model illustrates the use of slider parameters and port parameters by displaying interactively modified Lissajous curves.

The block diagram for the model was constructed using the Ptolemy II schematic editor called Vergil.

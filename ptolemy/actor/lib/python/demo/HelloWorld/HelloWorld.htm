HelloWorld Jython Applet

This applet does not use Ptolemy, it uses Jython.

HelloWorld.py To create HelloWorldApplet.jar we used the jythonc command, which is part of the Jython distribution.

In theory, if everything is installed properly, the following command should work: 

jythonc --core --deep --jar HelloWorldApplet.jar HelloWorld.py

However, we configured the makefile to run something like: 

CLASSPATH="../../../../../..;c:\ptII/lib/jython.jar" \
"/cygdrive/c/j2sdk1.4.2/bin/java" "-Dpython.home=../../../../../../vendors/jython/jython-2.1" \
	org.python.util.jython \
	../../../../../../vendors/jython/jython-2.1/Tools/jythonc/jythonc.py \
	--compiler javac \
	--core --deep --jar HelloWorldApplet.jar HelloWorld.py
For more information, see the Jython.org applets

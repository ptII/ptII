Ptolemnizer

The Ptolemnizer is an interactive model that converts text by adding a "p" to any word that begins with a "t" but not with a "th". Type a sentence below, such as "The Ptolemy tome takes time to read."

Ptolemnizer is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

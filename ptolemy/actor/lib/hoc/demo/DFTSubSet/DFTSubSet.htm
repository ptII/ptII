DFTSubSet

DFTSubSet is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

In this Higher Order Component (hoc) demonstration by Zoltan Kemenczy (Research in Motion), ComputeCoefficient contains a model to compute one DFT coefficient using the Goertzel kernel. NDFTCoefficients instances are used.

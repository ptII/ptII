ApplyFFT

ApplyFFT is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

This model illustrates the ApplyFunction and ApplyFunctionOverSequence actors. These apply a function defined in the expression language over either an individual input token or an array constructed from a sequence of input tokens.

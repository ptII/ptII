HocDE

HocDE is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

A simple example of the use of a higher-order component. The "Bank of N Counters" component will replicate its inside model N times and route each of N input channels of data to the corresponding replicate. In this case, the submodels simply count the events sent their way. The Switch actor randomly routes events to each of the N submodels.

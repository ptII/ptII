ConvolutionalCoder

ConvolutionalCoder is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

This example compares uncoded transmission of a binary signal through a channel that introduces random errors to transmission with an error-correcting code. The total number of errors in each case and error rate are displayed in the icons below. The ViterbiDecoder actor does hard decoding.

EightChannelFFT

This page contains an applet that should appear as a separate window.

To create your own FFT actor (in Ptalon), go to More Libraries... Ptalon and drag out a PtalonActor. Set its ptalonCodeLocation FileParameter to $PTII/ptolemy/actor/ptalon/demo/EightChannelFFT/FFT.ptln. You can then set the parameter N to some positive power of 2 to create the FFT.

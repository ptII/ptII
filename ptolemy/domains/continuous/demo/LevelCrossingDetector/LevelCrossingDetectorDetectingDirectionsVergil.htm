LevelCrossingDetectorDetectingDirections

This page contains an applet that should appear as a separate window.

Levels and directions detected by the level crossing detectors.

This model shows that level crossing detectors can be configured to detect different directions of level crossings with the "direction" parameter.

When the direction is configured to be "rising" (or "falling"), the level crossing detector only detects the level crossings when the input signal is rising (or falling). For example, the second (or third) level crossing detector.

By default, the direction is configured to be "both", meaning any level crossing will be detected no matter the signal is rising or falling.

StickyMasses

This model shows a hybrid system, which mixes continuous-time modeling with finite state machines. In this example, two point masses on springs oscillate. They occasionally collide, in which case, they stick together and oscillate together. The stickiness decays, and they eventually come apart again. This is an example of a modal model, where there are two modes, "together" and "separate". Each mode is modeled by a state in an FSM, and each state refines to a continuous-time model of the dynamics in that mode.

StickyMasses is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

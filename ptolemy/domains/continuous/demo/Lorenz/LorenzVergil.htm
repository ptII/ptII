Lorenz Attractor

\ \ MoML specification for this model. \

The applet above simulates these differential equations using the Continuous domain in Ptolemy II. The Continuous domain includes a sophisticated numerical solver for differential equations. The plot shows x2 vs. x1. The initial conditions of the states are all 1.0. The parameters sigma, lambda and b can be modified to change the behavior of the model.

See the model in Vergil, the Ptolemy II schematic editor.

ComparedClocks

This demo shows that different clocks have different behaviors in CT domain.

ComparedClocks is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details. The clocks have the same period as 0.5. The above two clocks have offsets as {0.0, 0.2}, while the bottom one has offsets as {0.1, 0.2}.

The two continuous clocks generate discontinuities at their offsets and the discontinuities are shown on the simulations results explicitly. While the discrete clock doesn't show that.

The block diagram for the model was constructed using the Ptolemy II schematic editor called Vergil.

HybridPlant

Hybrid model of a plant where a continuous flow of raw material is directed into bottles as the bottles (jobs) arrive. The top level is a discrete- event model with a modal continuous-time model inside. Model parameters are to the right.

HybridPlant is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

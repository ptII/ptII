ContinuousInsideDE

Combination of discrete-event modeling (at the top level) and continuous-time modeling (below that). This example models a job shop where job arrivals are discrete events, the processing rate is given by an exponential random variable, and the job processing is modeled in continuous time with an opaque composite actor.

ContinuousInsideDE is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

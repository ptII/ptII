Starmac

Vehicle Dynamics State: * Ve - Velocity, earth fixed frame (what the velocity appears to be to an outside observer, i.e., a vehicle heading North-East has its speed split [in an norm sense] between the x and y components) * Ab - Acceleration. * Xe - Position, earth fixed frame. * p,q,r - Angular velocity, body fixed frame. * DCM - Direction cosine matrix, a.k.a. rotation matrix. Rotates a vector between the body frame and the earth fixed frame. * Euler - The Euler angles.

Starmac is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

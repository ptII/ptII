NoiseSpectrum

This model measures the power spectrum of bandlimited noise and compares it against the theoretical value of the spectrum. Notice that the sidebands decay slightly more slowly than the theory predicts because of aliasing distortion. You can experiment with the bandwidth of the noise.

NoiseSpectrum is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

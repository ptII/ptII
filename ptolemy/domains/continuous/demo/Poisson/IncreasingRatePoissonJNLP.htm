IncreasingRatePoisson

This model is an example of a stochastic hybrid system. It generates a Poisson process with a linearly increasing rate using the Continuous domain. The model plots the events vs. time and a histogram of the time between events. The technique here was suggested by John Lygeros.

IncreasingRatePoisson is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

Bouncing Ball

This Continuous Time model uses a Finite State Machine and the GR domain to simulate a bouncing ball.

BouncingBall is available as:

  • An applet.

This continuous-time model uses a modal model to simulate a bouncing ball and the GR domain to animate it.

This is a classic example of a Zeno system, where an infinite number of bounce events occur in a finite time. Open the Ball Model to see how it is implemented.

SampledClocks

This model illustrates continuous-time signals and discrete signals. It shows that samples of a piecewise continuous signal have the value of the signal at the start of the discontinuity, as one would expect in hardware samplers that require a setup time. In the semantics of HyVisual (and continuous-time in Ptolemy II), piecewise continuous signals have more than one value at the time of the discontinuity. Discrete signals can also be sampled, but if the discrete signal has only one value at each discrete time, then the samples will reflect exactly that value. In this example, a triggered sampler is used to sample the discrete signal, and every second discrete clock output is sampled.

SampledClocks is available in two formats:

  • As a Web Start Application:
  • As an applet.

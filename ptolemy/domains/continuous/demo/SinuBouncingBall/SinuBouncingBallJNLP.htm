SinuBouncingBall

This model models a 3-D ball bouncing on a sinusoidal surface. Double-click the right yellow block for a graphic view. The ball collides with the surface and bounces with some energy loss, which is specified by the lambda parameter. The bouncing direction depends on the slope of the surface. There is no extra force except the gravity force applied on the ball. This model is a Zeno model. We use two states for pre-Zeno and post-Zeno dynamics respectively.

SinuBouncingBall is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

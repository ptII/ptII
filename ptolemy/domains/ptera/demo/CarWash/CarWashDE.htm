CarWashDE

CarWashDE is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

This CarWash system consists of multiple parallel servers sharing a single queue, at which customers arrive. The number of servers is specified in InitServers. The number of customers in the queue at the beginning of the simulation is specified in InitQueue. When the model is run, a plot appears that shows the number of available servers and the number of customers in line over time.

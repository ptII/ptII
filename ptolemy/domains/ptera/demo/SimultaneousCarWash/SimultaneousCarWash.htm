SimultaneousCarWash

SimultaneousCarWash is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

Look inside PteraModalModel to perceive its internal design, which contains a Simulate event with an Ptera refinement. The Simulate event is processed twice at model time 0, causing two instances of the refinement's initial event to be scheduled in the refinement's own event queue. The streams of events occurring in the two simulations are merged in that event queue and are processed in their time stamp orders.

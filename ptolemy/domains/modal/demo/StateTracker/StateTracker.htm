StateTracker

StateTracker is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

This is a very simple model by David Hermann, (Research in Motion) outputs which state it is in. It also demonstrates Transition Refinements by appending a string to the output in the refinements.

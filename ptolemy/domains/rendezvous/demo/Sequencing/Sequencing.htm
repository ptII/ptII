Sequencing

Sequencing is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

This demo compares two distinct ways to control the sequencing of otherwise independent tasks. The upper method is complex and subtle, and uses nondeterministic mechanisms to achieve deterministic aims. The lower method is simple and direct and uses only deterministic mechanisms.

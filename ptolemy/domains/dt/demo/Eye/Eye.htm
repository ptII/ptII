Eye is available in two formats:

The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

The plot shows the eye diagram for a received waveform for a baseband channel with additive Gaussian noise.

p

The default parameters model:

  • Additive Gaussian noise with standard deviation 0.1.
  • 100 % excess bandwidth square-root raised-cosine pulse.
  • Matched filter in the receiver.

The block diagram for the model was constructed using the Ptolemy II schematic editor called Vergil.

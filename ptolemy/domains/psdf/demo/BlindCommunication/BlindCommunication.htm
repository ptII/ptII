BlindCommunication

BlindCommunication is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

The AudioReader reads an input PSK signal and executes a sequence of signal analyses to determine the carrier, baud rate, and number of phase states. The resampler computes an unknown number of output samples, preventing static scheduling of the toplevel model. Since the amount of processed data changes, the PhaseStatesEstimator model cannot be a synchronous dataflow model. In this model the PhaseStatesEstimator is quasi-statically scheduled using the Parameterized Synchronous Dataflow domain.

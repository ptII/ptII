Model Display

The model below modifies a model produced by the ModelGenerator example in order to get animation. It creates MoML text to modify the colors of the circles.

ModelDisplay is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

Expression Evaluation

Expression is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

The slow signal is a ramp that starts at zero and increments by PI/100 on each firing. The fast signal is a ramp that starts at zero and increments by PI/10 on each firing. Try changing the expression in the box to "cos(slow)*cos(fast)" and clicking on the "Go" button. Other interesting alternatives include:

  • "cos(fast*cos(slow))"
  • "0.2*slow + cos(fast)"
For more information about the Ptolemy Expression language, see the Expression Language Documentation

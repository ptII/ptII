FourierSeries

The top plot represents one period of a periodic signal. The bottom plot displays the magnitude of the Fourier series coefficients for the periodic signal. In the top plot, you can right-click and drag to modify the plot. The bottom plot will be updated. Try sketching a constant signal, and note that the bottom plot will show that there is only a zero frequency term.

FourierSeries is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

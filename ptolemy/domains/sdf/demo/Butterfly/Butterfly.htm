Butterfly is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

This applet shows an elaborate curve called the butterfly curve, described by T. Fay, American Mathematical Monthly, 96(5), May, 1989. Above, the applet has been implemented directly in Java (see the source code).

Array

Array is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

Many operations on arrays are best done by converting the arrays to a sequences and back again. Here, a sequence of random numbers is converted to an array. An alternative would be to use a Const actor with the expression random(100).

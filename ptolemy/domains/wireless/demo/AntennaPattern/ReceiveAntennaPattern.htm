ReceiveAntennaPattern

This example illustrates modeling receiving antenna gain. The Receiver has an 8-element beamforming antenna with a steering angle applied. When you run this model, the Receiver will move around the Sender and measure the received power. It then plots the receive power on a polar plot, with the distance from the origin representing the received power, and the angle representing the angle of its position relative to the transmitter. In effect, it measures and displays the antenna gain pattern of the Receiver. (See TransmitAntennaPattern.xml as an example of modeling transmit antenna gain)

ReceiveAntennaPattern is available as:

  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

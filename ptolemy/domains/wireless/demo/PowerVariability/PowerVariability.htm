PowerVariability

PowerVariability is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

This model shows a transmitter (at the left) and a receiver (at the right), where the receiver moves as the model executes. The channel is a PowerLossChannel with a limited range. When the receiver is in range, the power depends on the distance to the transmitter, according to an inverse square law.

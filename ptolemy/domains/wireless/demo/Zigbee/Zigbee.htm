Zigbee

Zigbee is available as:

  • An applet.

This is a simple version of the protocol so that the simulation is as fast as possible. The model has the main characteristics of Zigbee, but without separate layers.

This version just implements a very simple Ad-Hoc On-Demand Distance Vector Routing (AODVR), the next step would be to implement the Medium Access based on CSMA-CA for non-Beaconing Networks.

FeedbackLoop

This model demonstrates that a NonStrictDelay actor breaks a feedback loop in an SR model.

FeedbackLoop is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

This model demonstrates that a NonStrictDelay actor breaks a feedback loop in a SR model, even when it has no initial value.

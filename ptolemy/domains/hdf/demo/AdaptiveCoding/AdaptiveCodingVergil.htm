AdaptiveCoding

This page contains an applet that should appear as a separate window.

This model demonstrates that HDF allows dynamic port rate changes and correctly schedules the two modal models.

In this example, the model has two modes of Hamming Codec, a (7, 4) Hamming Code and a (3, 1) Hamming code. Each set of Codec has different consumption rates and production rates. The Switch actor produces booleans to control the switching between the two coding/decoding schemes. Such switching signal might come from a sensor, a performance detector, or a command issued by a person.

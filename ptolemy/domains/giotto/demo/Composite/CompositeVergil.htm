Composite

This page contains an applet that should appear as a separate window.

This demo shows how a Giotto model works inside a DE model.

The trigger events generated from the event source actors simply update the inputs for the computation specified by the Giotto model. However, whether the inputs are consumed by the Giotto model is determined by the period of the Giotto model.

The Giotto model starts at the same time the DE model starts, and keeps running until its iteration counts are met or the top level (the DE model) stops execution.

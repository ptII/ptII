V2V

V2V is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

Vehicle-to-vehicle control system, as designed by U. Penn for Mobies. The controller in FollowingPlatoon is a modal controller with the objective of following the LeadingPlatoon.

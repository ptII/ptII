CTInsideDE

CTInsideDE is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

This model illustrates the use CT inside DE. The Ramp actor provides jobs to be processed by the Job Processor. The processing rate is random number (exponential). When a job has been processed it triggers another job from the Ramp. Note that this model could more easily be constructed using only DE, but the use of CT illustrates several features of this combination.

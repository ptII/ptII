IfThenElseFSM

IfThenElseFSM is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

This model illustrates how a Finite State Machine (FSM) Modal Model and a Dynamic Dataflow (DDF) model may be embedded in a Synchronous Dataflow (SDF) model.

See also a comparison between a Finite State Machine (FSM) implementation and a DDF implementation of this model.

IfThenElseFSM

IfThenElseFSM is available as:

  • An applet.

This model illustrates how a Finite State Machine (FSM) Modal Model and a Dynamic Dataflow (DDF) model may be embedded in a Synchronous Dataflow (SDF) model.

See also a comparison between a DDF implementation and the Finite State Machine (FSM) implementation of this model.

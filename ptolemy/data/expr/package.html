The Ptolemy II Expression Language.

The Ptolemy II Expression Language is documented in Chapter 3, "Expressions" (Ptolemy Website) Local HTML Version).

The expression language uses JavaCC, which is available from http://javacc.dev.java.net. You need only install JavaCC is you plan on extending the syntax of the expression language by modifying the .jjt files. $PTII/configure looks for JavaCC in $PTII/vendors/sun/javacc-3.2, so either install JavaCC in that location or use the --with-javacc= flag and then rerun configure.

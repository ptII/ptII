Sinewave

Double click to perform a custom analysis based on the lattice (described by the Custom attribute). The solver uses the explicit constraints along with the default constraints (set as its parameters) to resolve the property solution.

Sinewave is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

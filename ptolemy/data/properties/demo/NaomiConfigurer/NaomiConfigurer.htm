NaomiConfigurer

A simple model of the NAOMI framework. Each node, the blue circles, represents a domain-specific model created by some tool. These models share between each other a set of attributes or parameters set by a configurer (config).

NaomiConfigurer is available in two formats:

  • As a Web Start Application:
  • As an applet.
The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

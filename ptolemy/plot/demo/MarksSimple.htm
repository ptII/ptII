Simpler version of an applet that plots data read from a URL

This applet is similar to Marks.htm except that this applet uses the much simpler <applet> . . . </applet> style that will not work under Netscape Communicator 4.7x.

No Java Plug-in support for applet, see http://java.sun.com/products/plugin/

Use the View -> Source feature of your browser to view the source of this applet and compare and contrast it with the much more complex HTML in Marks.htm

For more information about how to write applets that use the Java Plug-in, see http://java.sun.com/products/plugin/versions.html

Note that because of a bug in Java 1.4.0_01, this applet may fail if it is being viewed on a local file system, see AccessControlException caused by spaces in path name for details.

Code Generation Demonstrations Listed by Top-Level Domain

There are two classes of Codegen demos:
  • Those demos that do not use the plotter, and thus will work with a Java Runtime Environment (JRE), such as the JRE that is shipped with the Windows installer.
  • Those demos that do use the plotter, and thus require that jvm.dll be in the path. The Ptolemy code generator makes every effort to set your path. Currently, the plotter demos only work under Windows.

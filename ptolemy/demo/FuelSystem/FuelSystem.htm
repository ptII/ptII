FuelSystem

Model with a single Tank. The tank has a constant input flow of 1.0. The output flow 0.0 between time 0 and time 3 and it is 2.0 between time 3 and time 8. At time 8 the output flow is changed to 0.0 again and the sequence of values for the output flow is repeated. In this example, the signal starts chattering when the tank becomes empty. This is due to a modeling error.

FuelSystem is available in two formats:

  • As a applet.

The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

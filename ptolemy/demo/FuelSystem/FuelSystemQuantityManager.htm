FuelSystemQuantityManager

This model draws fuel at a constant rate from Tank1. When the level in Tank1 gets below 1/4 of its capacity, the model begins transferring fuel from Tank2 to Tank1 (while it continues to draw fuel from Tank1 at a constant rate).

The FuelMover receives inputs through a network component, the Bus. The Bus has a parameter for the service time, i.e. the time it takes to transfer a message on the network. This service time has an influence on the behavior of the model as messages get delayed.

FuelSystemQuantityManager is available in two formats:

  • As a applet.

    • The Java Web Start version is preferred, though browsers with versions of Java earlier than Java 1.6.0_13 may only work with the applet version. See Web Start Help for details.

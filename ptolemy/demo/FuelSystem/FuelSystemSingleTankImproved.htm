FuelSystemSingleTankImproved

Improved model with a single Tank. Like the chattering model, this one limits the input and output flows when the tank becomes full or empty, but it does so without introducing chattering behavior. This is a modal model that leverages the rigorous Ptolemy II semantics for hybrid systems. See the two-tank model for a more interesting test case for this tank model.
